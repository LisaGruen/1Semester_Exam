01.04.01 Responsive website

Task description

In this module we were introduced to basic web design and design principles. We learned about html and CSS, Adobe Photoshop and Illustrator and how to build a responsive website. Additional, we learned how to test our product in terms of usability with for example the 5 seconds test. Our task was to build a responsive website for a given art style. In my case it was the art style was Postmodernism, but because this period is very broad I narrowed it down to “Feminist Art in Postmodernism”.

What I learned

Learning about the design principles like Gestalt laws and contrasts helps to reflect and improve your decisions during the work process and to communicate your design. Using a FTP-client and Brackets for the first time and writing your first html and CSS lines were exciting and overwhelming in the same moment.

What I would change today

Using grids would make it easier to create a responsive page. I would change the headlines from justified text to centered text in the media query for the mobile version. The design page needs another structure for a better overview. One solution could be a grid to divide the content with the code and viusally with for example boxes.
